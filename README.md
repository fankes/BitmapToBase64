# BitmapToBase64

A simple Tools.

## 用法

- 编译并安装 APK

- 手动授予存储权限(Android 11 及以上系统还需要授予所有文件访问权限)

- 将需要处理的图片放到 /内置存储卡/Documents/BitmapToBase64/orig.png

- 点击“开始工作”按钮

- 处理后的文件位于 /内置存储卡/Documents/BitmapToBase64/out.txt

- 打开 out.txt 得到 Base64 内容
